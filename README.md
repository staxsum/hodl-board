# HODL Board

A simple, clean cryptocurrency market dashboard with real-time data and analytics.

## Features

- Live price ticker for 50+ cryptocurrencies
- Interactive charts for major crypto pairs
- Fear & Greed Index for market sentiment
- Crypto profit calculator
- Market bubble visualization
- Curated watchlists: US Tokens, Memes, DeFi, RWA, DePin, AI

## Quick Start

1. Download or clone this repo
2. Open `index.html` in your browser
3. That's it! All data loads automatically

No installation, no API keys, no backend required.

## Customize

**Change tokens in watchlists:**
Edit `coin-ids` in the HTML file. Find coin IDs at [CoinGecko](https://www.coingecko.com/)

```html
coin-ids="bitcoin,ethereum,solana"
```

**Change chart symbols:**
Edit the `symbols` array in the TradingView widget

## Mobile Friendly

Fully responsive - works great on desktop, tablet, and mobile.
